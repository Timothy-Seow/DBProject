## How to run
1) Go into the web directory: cd web
2) Run program: python app.py
3) Alternatively, run: python web/app.py

## Inserting data into DB
1) Dataset is too large to be included, it can instead be found at: https://www.kaggle.com/datasets/ahmedshahriarsakib/uber-eats-usa-restaurants-menus
2) Make sure pandas is installed: pip install pandas
3) Run the data loader: python dataLoader.py
