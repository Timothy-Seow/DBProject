## How to run dataloader
1) As dataset is too large to be included, download it from: https://www.kaggle.com/datasets/ahmedshahriarsakib/uber-eats-usa-restaurants-menus
2) Make sure pandas is installed: pip install pandas
3) Run the data loader: python dataLoader.py

## Notes
- As the generated DB file is included, it is not necessary to run the data loader unless more data is needed
